# Supply Chain Performance Project

## 📌 Overview

This project focuses on analyzing supply chain performance using data analysis and visualization techniques. The goal is to identify key insights that help improve operational efficiency, delivery performance, and overall business decision-making.



## 🛠️ Tools & Technologies

* Python (Pandas, NumPy, Matplotlib, Seaborn)
* SQL
* Power BI



## 🐍 Python Analysis

In this project, Python was used for:

* Data cleaning and preprocessing
* Handling missing values
* Creating new calculated fields
* Exploratory Data Analysis (EDA)
* Generating visualizations to understand patterns and trends

### Key Visualizations:

* Sales trends over time
* Order distribution by region
* Product performance analysis
* Delivery performance insights



## 🗄️ SQL Analysis

SQL was used to:

* Query and aggregate data
* Analyze performance metrics
* Extract insights such as:

  * Total sales by region
  * Top-performing products
  * Order frequency and trends



## 📊 Power BI Dashboard

An interactive dashboard was created using Power BI to visualize key metrics:

### Dashboard Highlights:

* Total Sales & KPIs
* Sales by Region
* Top Products
* Order Trends Over Time
* Delivery Performance

The dashboard helps stakeholders quickly understand business performance and make data-driven decisions.



## 📈 Key Insights

* Identified top-performing regions and products
* Detected patterns in order trends
* Highlighted areas with delivery delays
* Provided actionable insights to improve supply chain efficiency



## 📂 Project Structure

* Python scripts for data processing and visualization
* SQL queries for data extraction and analysis
* Power BI dashboard for reporting
* Dataset used for analysis



## 🚀 Conclusion

This project demonstrates how combining Python, SQL, and Power BI can provide powerful insights into supply chain operations and support better decision-making.
